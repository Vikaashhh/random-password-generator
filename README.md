# ==========================================================
# 🔐 Random Password Generator
# ==========================================================
# A minimal, responsive, and easy-to-use random password
# generator built using HTML, CSS, and JavaScript. This 
# utility helps users create strong and secure passwords 
# instantly with a single click.

# ==========================================================
# ==========================================================
# 📌 Features
# ----------------------------------------------------------
# ✅ Instantly generate strong random passwords
# ✅ Copy password to clipboard functionality
# ✅ Responsive layout and clean UI
# ✅ Built with pure HTML, CSS, and JavaScript
# ✅ No external dependencies

# ==========================================================
# 🚀 Tech Stack
# ----------------------------------------------------------
# - HTML
# - CSS
# - JavaScript

# ==========================================================
# 🛠️ How to Use
# ----------------------------------------------------------
# 1. Clone the repository
```sh
git clone https://github.com/Vikaashhh/random-password-generator.git
```

# 2. Navigate into the project directory
```sh
cd random-password-generator
```

# 3. Open the index.html file in your browser
```sh
start index.html        # On Windows
open index.html         # On macOS
xdg-open index.html     # On Linux
```
